# Projeto-Letmeask

Sobre o projeto: perfeito para criadores de conteúdos poderem criar salas de Q&A com o seu público, de uma forma muito organizada e democrática.

### Esse projeto foi desenvolvido com as seguintes tecnologias:

* React
* Firebase
* TypeScript
